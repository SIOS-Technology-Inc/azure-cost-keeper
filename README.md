## Azureの使いすぎを防止するOSS「Azure Cost Keeper」
このOSSは、Azureの使いすぎによる意図せぬ課金を防止するための様々な機能を提供します。

- リソースグループごとの詳細な課金レポート(利用料、利用者、月末時点での予測料金など)
- 不要なリソースの自動停止(リリース予定)
- 不要なリソースグループの棚卸し機能(リリース予定)

Dockerイメージでも提供されており、すぐに実行することが可能です。
